The class must:
1. Accept `pool: asyncpg.Pool` in `__init__`
2. Implement these async methods using `async with pool.acquire() as conn`:
   - `create(title: Optional[str], mode: str) -> dict`: INSERT into sessions, RETURNING all columns. Auto-generate UUID hex (8 chars) for id.
   - `get(session_id: str) -> Optional[dict]`: SELECT * FROM sessions WHERE id = $1
   - `list() -> list[dict]`: SELECT * FROM sessions ORDER BY created_at DESC
   - `delete(session_id: str) -> bool`: DELETE FROM sessions WHERE id = $1, return whether a row was deleted

All methods must:
- Use parameterized queries (`$1`, `$2`) — NO string interpolation
- Convert rows to dicts via `dict(row)`
- Use `uuid.uuid4().hex[:8]` for session ID generation

Example create():
```python
async def create(self, title: Optional[str], mode: str) -> dict:
    import uuid
    async with self.pool.acquire() as conn:
        row = await conn.fetchrow(
            """
            INSERT INTO sessions (id, title, mode)
            VALUES ($1, $2, $3)
            RETURNING id, title, mode, created_at, updated_at
            """,
            uuid.uuid4().hex[:8],
            title,
            mode,
        )
        return dict(row)
```

The class must:
1. Accept `pool: asyncpg.Pool` in `__init__`
2. Implement these async methods:
   - `create(session_id: str, role: str, content: str, metadata: dict = {}) -> dict`: INSERT into messages, RETURNING all columns. Store citations and usage in the `metadata` JSONB field.
   - `get_recent(session_id: str, limit: int) -> list[dict]`: SELECT role, content FROM messages WHERE session_id = $1 AND role != 'system' ORDER BY created_at DESC LIMIT $2. Then reverse to chronological order (oldest first). Default limit comes from `settings.chat_history_window`.
   - `create_system(session_id: str, content: str) -> dict`: INSERT with role='system', returning dict.

For `get_recent`, the query must:
- Exclude system messages (`role != 'system'`) so system prompt is never counted in the window
- Order by `created_at DESC` then reverse to get chronological order
- Return list[dict] with `{"role": ..., "content": ...}`

```python
async def get_recent(self, session_id: str, limit: int) -> list[dict]:
    async with self.pool.acquire() as conn:
        rows = await conn.fetch(
            """
            SELECT role, content FROM messages
            WHERE session_id = $1 AND role != 'system'
            ORDER BY created_at DESC
            LIMIT $2
            """,
            session_id,
            limit,
        )
        # Reverse to chronological order (oldest first)
        return [dict(r) for r in reversed(rows)]
```

Note: Per D-03, the window is configurable via CHAT_HISTORY_WINDOW. The `get_recent` method accepts `limit` as a parameter so the caller can pass `settings.chat_history_window`.
