<objective>
Fix message layout overflow bugs across the chat interface. Apply responsive max-width, fix flex overflow, and ensure word-wrapping for all screen sizes.

Purpose: Eliminate the root cause of "messages that stretch out" — no more broken layouts on long URLs, code blocks, or narrow mobile screens.
Output: Updated MessageList.tsx, ChatShell.tsx (header layout), and Composer.tsx with proper overflow handling.
</objective>

<task type="auto">
  <name>Task 1: Fix MessageList overflow (D-01)</name>
  <files>apps/frontend/src/components/MessageList.tsx</files>
  <action>
Read MessageList.tsx first.

Apply these changes to MessageList.tsx:

1. **Outer container (line ~20):** Change `div className="flex-1 overflow-y-auto space-y-4 p-6"` to:
   ```
   div className="flex-1 overflow-y-auto overflow-x-hidden space-y-4 p-6"
   ```
   This prevents horizontal overflow from bubbling up.

2. **Message bubble wrapper (line ~33):** Change `div className={`flex ${msg.role === 'user' ? 'justify-end' : 'justify-start'}`}` to:
   ```
   div className={`flex ${msg.role === 'user' ? 'justify-end' : 'justify-start'} min-w-0`}
   ```
   This fixes the flex overflow that stretches the container.

3. **Message bubble (line ~38):** Update `max-w-[70%]` to use responsive variants:
   ```
   div className={`max-w-[70%] md:max-w-[90%] lg:max-w-[70%] rounded-2xl px-4 py-3 ${...}`}
   ```
   - Mobile (<768px): 100% width
   - Tablet (768-1023px): 90% width
   - Desktop (≥1024px): 70% width

4. **Message text content:** Add `overflow-wrap: break-word` class. The prose wrapper handles most cases, but for raw text paragraphs inside MessageContent, ensure:
   ```
   p className="whitespace-pre-wrap wrap-break-word leading-relaxed"
   ```

5. **Citation section container (inside message bubble, after citations):** Add `overflow-x-auto` to prevent citation grid from causing horizontal scroll.

All changes must use Tailwind classes only — no inline styles.
  </action>
  <verify>
grep -n "overflow-x-hidden\|min-w-0\|max-w-\[70%\]\|max-w-\[90%\]" apps/frontend/src/components/MessageList.tsx
# Must find: overflow-x-hidden on outer container, min-w-0 on flex wrapper, responsive max-w on bubble
  </verify>
  <done>
Message bubbles respect responsive breakpoints (70%/90%/100%), containers prevent horizontal overflow, URLs and code blocks wrap within bubbles.
</done>
</task>
