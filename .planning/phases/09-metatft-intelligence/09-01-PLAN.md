Create `apps/backend/scripts/scrape_metatft.py` following the scrape_patch17.py pattern.

**Scraping (per D-01, D-02):**
1. Use `httpx` to fetch `https://www.metatft.com/comps` with User-Agent header (same as scrape_patch17.py)
2. Extract embedded JSON from page HTML using regex: `re.findall(r'window\.__INITIAL_STATE__\s*=\s*(\{.*?\})', html, re.DOTALL)` or `re.search(r'"comps":\s*(\[.*?\])', html, re.DOTALL)` — test which pattern works on the page
3. Parse the JSON to extract comp data: comp name, tier (S/A/B), top4 rate, carry champion, items, traits

**Vietnamese Markdown Transform (per D-03):**
For each comp, create a markdown chunk with this format:
```
Theo MetaTFT hien tai, Dong hinh **[{comp_name}]** dang o Tier **{tier}** voi **{top4_rate}%** ti le Top 4.
Tuong Carry la **{carry}** voi **items**: {items_list}.
Traits: {traits_list}.
```

**Metadata:**
```python
metadata = {
    "season": "SET17",
    "patch": patch_version or "latest",
    "type": "metatft_comps",
    "source": "metatft",
}
```

**Ingest pattern (per D-04):**
- source = `metatft:comps:{comp_name_slug}`
- Use `content_hash()` from `app.utils.hashing`
- Use `ollama.generate_embeddings()` from `app.services.ollama`
- Insert into `chunks` table with `ON CONFLICT (source, content_hash) DO NOTHING`
- Truncate embeddings to 1024 dims (MAX_EMBEDDING_DIMS = 1024)

**Public API:**
