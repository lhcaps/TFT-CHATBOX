**CompCard Markdown Syntax to parse:**
```markdown
### Comp: {Comp Name}
**Tier:** {S/A/B} | **Top4:** {X%} | **Avg Place:** {X.X}
**Carry:** {Champion Name}
**Items:** [{Item1}] [{Item2}] [{Item3}]
**Units:** {Unit1}, {Unit2}, {Unit3}, {Unit4}, {Unit5}, {Unit6}
**Traits:** {Trait1} {N}, {Trait2} {M}
```

**Component Interface:**
```typescript
interface CompCardProps {
  compName: string;
  tier: 'S' | 'A' | 'B';
  top4Rate: string;
  avgPlace: string;
  carry: string;
  items: string[];
  units: string[];
  traits: { name: string; count: number }[];
}
```

**Tier Badge Colors:**
- Tier S: Gold (`#fbbf24`) with dark text
- Tier A: Silver (`#9ca3af`) with dark text
- Tier B: Bronze (`#b45309`) with light text

**Item Color Coding (based on item type):**
- Attack Damage items: Red tint (`rgba(239,68,68,0.15)` border)
- AP items: Blue tint (`rgba(59,130,246,0.15)` border)
- Tank items: Green tint (`rgba(34,197,94,0.15)` border)
- Support items: Purple tint (`rgba(168,85,247,0.15)` border)

Map common items to categories:
- AD items: Giant Slayer, Bloodthirster, Infinity Edge, Rageblade, Runaan's, etc.
- AP items: Qujin, Rabadon's, Morellonomicon, etc.
- Tank items: Warmog's, Bramble Vest, Gargoyle's, etc.
- Support items: Redemption, Zeke's, Locket, etc.
Items not in the list: neutral gray (`rgba(156,163,175,0.15)` border).

**Layout:**
```
┌─────────────────────────────────────────────────────┐
│ ### Comp: Katarina Duelist                    [S]  │
├─────────────────────────────────────────────────────┤
│ Tier: S  |  Top4: 52%  |  Avg Place: 2.1         │
├─────────────────────────────────────────────────────┤
│ Carry: Katarina                                    │
├─────────────────────────────────────────────────────┤
│ Items: [GS] [BT] [QSS]                            │
├─────────────────────────────────────────────────────┤
│ Units: Zed, Ekko, Riven, Fiora, Camille, Jax       │
├─────────────────────────────────────────────────────┤
│ Traits: Duelist 6  |  Arcane 3                    │
└─────────────────────────────────────────────────────┘
```

**Exported function:**
```typescript
export function CompCard({ compName, tier, top4Rate, avgPlace, carry, items, units, traits }: CompCardProps): JSX.Element
```

Also export a parsing function for the markdown:
```typescript
export function parseCompCard(markdown: string): CompCardProps | null
```

The parser should extract fields from the CompCard markdown block using regex.

Use inline styles consistently (like PatchStatus.tsx). Do NOT use Tailwind classes for this component — follow the existing pattern.
